Listas Ordenadas: <ol> / <li>

Criando listas com ol:

  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5

A tag ol possui um parametro type
onde se pode informar:
1- (valor padrao)
A- (lista alfabética Maiuscula)
a- (lista alfabetica minuscula)
i- (lista romanos minisculo)
I- (lista romanos Maiuscula)

Exemplos

type = "I"

  1. item 1
  2. item 2
  3. item 3

type = "A"

  1. item 1
  2. item 2
  3. item 3

tyoe = "a"

  1. item 1
  2. item 2
  3. item 3

Listas não Ordenadas: <ul> <li>

Para criar uma unordered list, vamos usar a tag ul para delimitar a lista e a tag li para criar cada um dos itens internos.
Podemos alterar o parametro type com os valores:
disc - padrão, bola preta preenchida
circle - bola com borda preta e sem preenchimento
square - quadrado preto preenchido
